# Enonic: the CMS that adapts to your front-end

Enonic's modular and headless architecture combines ease of use and flexibility in a platform built for mission critical deployments. Build your content model, customize the authoring interface, run server functions, and augment the API.

[Get started for free](https://developer.enonic.com/start) • [Tutorials](https://developer.enonic.com/learn) • [Community](https://slack.enonic.com) • [Enonic Market](https://market.enonic.com) • [Enonic.com](https://enonic.com)

## Why developers choose Enonic

*   **Content management**: Seamlessly combine headless GraphQL delivery with powerful page building and visual editing in Content Studio.
*   **Rich schema system**: Define your content models in YAML, keep them in version control, and get instant API updates.
*   **NoSQL storage**: A built-in, high-performance NoSQL repository for structured content and binary assets.
*   **Customize the back-end**: Build functions, controllers, tasks, and APIs directly on the platform using a modern JS/TS environment.
*   **Scalable & secure**: ISO-27001 certified and designed for high-traffic, mission-critical enterprise applications.
*   **Open source**: Enonic is fully open source with support and SaaS options.

## What you'll get

*   **Content Studio**: A visual, intuitive authoring interface that editors love.
*   **Guillotine**: A high-performance, customizable headless GraphQL API.
*   **Enonic CLI**: A powerful tool for managing instances, building apps, and deploying to the cloud.
*   **Extensible library**: A vast ecosystem of starters, libraries, and apps on Enonic Market.
*   **Framework agnostic**: Use Next.js, React, Svelte, Remix, Astro, or Vue

## Quickstart

```bash
npm install -g @enonic/cli

enonic sandbox create mysandbox -t essentials -f
```

Then follow the intro guide: https://developer.enonic.com/learn/intro/stable/setup

You now have a fully working CMS with:

*   GraphQL API
*   Content storage
*   Demo content types and data
*   Content Studio authoring interface
